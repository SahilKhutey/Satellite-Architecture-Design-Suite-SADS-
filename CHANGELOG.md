# Changelog

All notable changes to SADS will be documented in this file.

## [1.0.0] - 2026-06-13
### Added
- Monorepo structural transition for scalability.
- 8 core subsystem physics engines (ADCS, Comm, Orbit, Power, Propulsion, Thermal, Structures, Payload).
- 13 microservices skeletons and gateway configuration.
- SysML modeling and Requirements trace folders.
- Docker-compose orchestration for TimescaleDB and Redis.
- Documentation reorganization.
