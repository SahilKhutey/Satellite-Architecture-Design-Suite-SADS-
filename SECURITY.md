# Security Policy

## Product Security Standards
The Satellite Architecture Design Suite (SADS) is designed for aerospace digital engineering. Ensuring model integrity, telemetry security, and system safety is a core priority.

## Supported Versions
Only the latest release version on the `main` branch is actively supported with security updates.

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability
Do not open a public GitHub issue for security vulnerabilities. Instead, email security@sads.aero with a description of the issue, steps to reproduce, and potential impact.

We will acknowledge receipt within 48 hours and provide updates regarding a fix.
