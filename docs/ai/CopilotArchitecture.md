# SADS AI Copilot Reference — Copilot Architecture

**Document ID:** SADS-AI-COP-001  
**Revision:** 1.0  
**Classification:** Engineering Reference

---

## 1. Overview

The AI Engineering Copilot assist systems designers by reviewing margin calculations, running sizing optimizations, and checking compliance logs.

---

## 2. Architecture Layout

* **Architecture Advisor**: Audits power, thermal, and pointing error budgets.
* **Aerospace RAG**: Embeds reference literature (Wertz SMAD, NASA-STD) to supply direct, traceable citations.
* **Optimization Agent**: Sizers components according to spacecraft bounds.
