## 1. Overview

SADS supports block definition diagrams and interface definitions compatible with SysML concepts.

---

## 2. Modeling Block Types
* **Blocks**: Components representing physical subsystems (e.g. Bus, Solar Panel, Battery).
* **Ports**: Input/Output interfaces with flow types (Power, Data, Thermal, Fluid).
* **Connectors**: Associations mapping flow directions between ports.
