# SADS Simulation Reference — Validation Framework

**Document ID:** SADS-SIM-VAL-001  
**Revision:** 1.0  
**Classification:** Engineering Reference

---

## 1. Overview

SADS provides a high-fidelity comparative validation framework. Calculations are checked against real-world satellites and historic telemetry datasets (e.g. Hubble, ISS).

---

## 2. Validation Standards
Calculations must match NASA and ESA references:
* **Level 1 (Concept Validation)**: Accuracy within $\pm20\%$.
* **Level 2 (Engineering Validation)**: Accuracy within $\pm10\%$.
* **Level 3 (High Fidelity Validation)**: Accuracy within $\pm5\%$.
* **Level 4 (Mission Grade Validation)**: Accuracy within $\pm1\%$.
  * Ex. ISS orbital period must be calculated to within 1% of the $92.8\text{ min}$ historical average.
