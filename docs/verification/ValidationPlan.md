# Validation Plan

## 1. Scope
Validates SADS outputs against industry tools (such as GMAT and STK) and historical spacecraft flight logs.

## 2. Comparative Benchmarks
- **Orbit Propagation:** Compare J2 regression rates against GMAT Keplerian Propagator.
- **Thermal Solver:** Validate transient temperature curves against reference thermal network datasets.
- **Link Budgets:** Compare path loss outputs with NASA Link Budget templates.
